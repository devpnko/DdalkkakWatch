⌚ 무화면 받아쓰기 — Knock-Knock v0.2

워치 화면을 아예 안 켜고 손목을 두 번 두드리면(Knock-Knock) 받아쓰기가 켜져요. 워치 본질에 가장 가까운 사용법. 모든 워치 path(C1~C5)에 공통.

원리 — 왜 화면 안 켜도 되나

  • Galaxy Watch의 Knock-Knock(손목 두 번 두드림) 손동작에 "딸깍 받아쓰기"를 할당
  • 두드리면 화면 안 뜨고(투명) Opt+Cmd+X를 Mac에 즉시 발사 → 받아쓰기 토글
  • 백그라운드 서비스가 BLE 연결을 미리 유지 → 두드리자마자 작동 (지연 없음)

전제 — C1 셋업(워치↔Mac BT 페어링 + 단축키 Opt+Cmd+X) 완료. → C1 먼저

1) 워치에 백그라운드 연결 켜기

  1. 워치에서 딸깍 앱을 한 번 실행 (이때 BLE 연결 유지 서비스가 자동 시작)
  2. 알림에 "딸깍 연결 유지 중"이 뜨면 정상 — 이제 앱을 닫아도 연결 유지됨

2) Knock-Knock에 "딸깍 받아쓰기" 할당

  1. 워치 → 설정 → 고급 기능 → 손동작 (또는 제스처)
  2. Knock-Knock(노크노크 / 두드리기) 켜기
  3. 실행할 앱/단축키 → "딸깍 받아쓰기" 선택
    앱 서랍에 "딸깍 받아쓰기"가 본체 앱과 별도로 보입니다 (무화면 전용 진입점)

3) 사용 + 검증

  1. 워치 화면 끈 채로 손목을 똑똑 두 번 두드림
  2. 햅틱 진동(두 번) = 받아쓰기 ON 신호
  3. "안녕하세요" 발화 → Mac 텍스트 필드에 입력
  4. 다시 두드림 = 받아쓰기 OFF (OpenTypeless가 toggle 모드일 때)

첫 두드림이 안 되면 — BLE가 cold 상태. 한 번 더 두드리면(롱 진동 후) 연결되고, 그다음부터 즉시 작동.

⚠️ 한계 / 주의

  • OpenTypeless / Superwhisper 단축키가 tap-to-toggle 모드여야 한 번 두드림으로 ON/OFF. hold 모드면 화면 PTT(꾹 누름)를 쓰세요.
  • 운전·요리·운동 중 손목 움직임에 오작동 가능 → 안 쓸 땐 Knock-Knock 끄기
  • Galaxy Watch Ultra/Watch7+ One UI 6.1.1+ 에서 Knock-Knock 지원

💡 다른 무화면 옵션 (참고)

  • 손목 휘두름(Wrist Flick) — 앱 켜져 있을 때만. 화면 PTT 보조
  • Double Pinch(엄지+검지 두 번) — Galaxy 시스템 전용이라 현재 앱 직접 연동 불가 (Samsung 미개방). 그래서 Knock-Knock으로 우회.
