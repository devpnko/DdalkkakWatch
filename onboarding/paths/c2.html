C2 — Galaxy Buds / AirPods + 워치 PTT 지원

이어폰 마이크가 입에 가까워 음질이 좋아. Mac에서 5m 떨어져도 OK (BT만 닿으면).

전제

C1 셋업이 끝나 있어야 함. (워치 ↔ Mac Bluetooth HID 연결됨, 단축키 Opt+Cmd+X로 설정됨)

→ C1 가이드 먼저 보기

1) 이어폰을 Mac에 페어링

  • Galaxy Buds: 케이스 열고 버튼 길게 → 깜빡임 → Mac BT에서 검색 → 연결
  • AirPods: 케이스 뒷면 버튼 길게 → Mac BT에서 검색 → 연결
  • 핵심 — 이어폰을 워치가 아니라 Mac에 페어링해야 함

2) Mac 마이크 입력 변경

  1. 시스템 설정 → 사운드 → 입력 → Galaxy Buds (또는 AirPods) 선택
  2. 입력 레벨 확인 — 안 잡히면 BT 끄고 켜기
  3. HFP 모드로 자동 전환됨 (음악 음질 ↓, 마이크 ON)

3) 받아쓰기 앱 마이크도 동일하게

  • OpenTypeless → Preferences → Microphone → Galaxy Buds
  • Superwhisper → Preferences → Audio → Galaxy Buds

4) 작동 확인

  1. 이어폰 착용
  2. 아무 텍스트 필드 커서
  3. 워치 꾹 누름 → "안녕하세요" → 손 떼기 → 입력

참고 — 이어폰 마이크가 활성화되면 음악이 잠시 끊김 (HFP 자동 전환).
