C3 — 외부 USB-C 마이크 + 워치 PTT 권고

유튜브 녹화·줌 회의·인터뷰 등 결과물 품질이 중요한 경우. 음질 9/10.

전제 + 장비

  • C1 셋업 완료 (워치 ↔ Mac BLE HID)
  • 외부 마이크 (택1):
    • Hollyland Lark M2 — 15만 원대, 무선 옷깃 마이크
    • DJI Mic 2 — 25만 원대, 듀얼 채널
    • Rode Wireless ME — 20만 원대
    • 유선 옵션 — Rode SmartLav+ (5만 원대)

1) 마이크 → Mac 연결

  • USB-C 직결 (가장 안정) — 송신기를 USB-C로 Mac에
  • 3.5mm — Mac에 헤드폰 잭 있으면 (M1 이상 일부)
  • UAC 모드로 인식되면 driverless 작동

2) Mac 사운드 입력 변경

  1. 시스템 설정 → 사운드 → 입력 → Lark M2 (또는 DJI Mic 등) 선택
  2. 입력 레벨 강하게 — 게인 조정 필요시 마이크 송신기에서

3) 받아쓰기 앱 입력도 동일하게

  • OpenTypeless / Superwhisper 각각의 마이크 설정에서 외장 mic로

4) 검증 + 활용

  1. 마이크 옷깃에 부착 (입에서 20cm)
  2. 워치 꾹 누름 → 발화 → 손 떼기 → 입력
  3. 회의 중에는 Mac 마이크는 보조, 외장은 발화 → 받아쓰기에만

팁 — 회의 녹화 중에도 워치 PTT는 동시에 작동. 회의 음성 ≠ 받아쓰기 음성으로 분리 가능 (입력 라우팅 설정).
