C6 — Mac 단축키 단독 (워치 없음) 지원

워치가 없거나, 워치는 다른 용도로 쓰고 싶을 때. Mac 키보드 단축키 1발.

왜 이 path

  • 워치 살 필요 없음
  • 키보드 위 손 자세 그대로 받아쓰기
  • 워치에서 오는 추가 가치 (손이 키보드 떠나도 OK)는 없음 — 그게 필요하면 C1~C5

1) 받아쓰기 앱 설치

  • OpenTypeless — 오픈소스, 한국어 좋음 (추천)
  • Superwhisper — 유료, 정확도 높음
  • macOS 받아쓰기 — 시스템 내장, 무료
  • Wispr Flow / Voibe / Aiko — 상용 옵션

2) 단축키 설정

본인이 편한 조합으로. Spotlight과 충돌 X가 핵심.

  • Opt+Space — Spotlight 대체
  • Opt+Cmd+X — 워치 path와 같음 (워치 추가시 호환)
  • F19 / F20 — 외장 키보드에 있으면

3) 마이크 입력

  1. 시스템 설정 → 사운드 → 입력 → 원하는 마이크
  2. 외장 mic 있으면 외장이 정확도 ↑

4) 작동 확인

  1. 아무 텍스트 필드
  2. 단축키 → 받아쓰기 UI
  3. 발화 → 텍스트
