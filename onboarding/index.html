손목을 두 번 두드리면(Knock-Knock) 화면 없이 받아쓰기가 켜져요. 워치 본질에 가장 가까운 사용법.

→ 무화면 받아쓰기 셋업

증상별

  • 워치 상태가 "HID 등록 대기" 그대로 — 워치 BT 권한 재허용 + 앱 재시작
  • Mac에서 워치가 안 보임 — 워치 앱에서 한 번 누르면 2분간 discoverable, 그 사이 Mac BT 설정에서 페어링
  • 입력은 되는데 받아쓰기 안 뜸 — Mac의 OpenTypeless / Superwhisper / macOS 받아쓰기 단축키가 Opt+Cmd+X인지 확인
  • 한 번 누르면 "ㅁ"으로 입력 — Mac 키보드 입력 소스가 한글 → 영문(US)으로 전환 필요
  • Mac 잠금 풀면 끊김 — 워치 앱 다시 열어 자동 재연결, 또는 Mac BT 끄고 켜기
  • adb로 워치가 안 잡힘 / 태블릿이 잡힘 — adb devices -l로 model 확인. 워치 무선 디버깅 다시 켜고 adb pair → adb connect. sleep 시 끊기니 화면 깨우기
  • 설치 시 "missing shared library wearable" — 설치 대상이 워치가 아님(태블릿/폰). 워치만 연결 후 재설치
